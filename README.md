# electron-vue-demo
A demo for "electron + webpack + vue"


## 安装
```shell script
npm install 
# 或 
cnpm install
```

## 开发
```shell script
npm start 
# 或 
npm run dev
```

## 编译
需要准备好代码签名，参考：https://www.electron.build/code-signing
```shell script
npm run build # 生产环境
npm run build-test # 测试环境 
```
